# Summarizer

A helpful skill that summarizes long documents into key points.

## Usage

Invoke with `/summarize` followed by the text you want summarized.

## Examples

```
/summarize [paste your document here]
```

```
/summarize Please summarize my meeting notes from today
```

## What It Does

- Extracts key sentences from long text
- Returns a concise 2-3 sentence summary
- Works with any text content
